# 小灵同学看板系统 - 轻量运行版

## 系统简介
这是一个即开即用的看板系统运行包，无需复杂配置，一键启动即可使用。

## 快速开始

### Windows用户
1. 双击运行 `start_kanban.bat`
2. 按照提示操作
3. 在浏览器中访问：http://localhost:5000

### 其他系统用户
1. 打开命令行终端
2. 进入本目录
3. 运行：`python kanban_enhanced.py`

## 系统要求
- Python 3.8或更高版本
- 网络连接（用于手机访问）

## 一键启动脚本功能

### start_kanban.bat (Windows批处理)
- 自动检测Python环境
- 自动安装所需依赖
- 显示访问地址
- 自动启动服务器

### start_kanban.ps1 (PowerShell脚本)
- 更强大的环境检测
- 自动获取本机IP地址
- 更好的错误处理
- 彩色输出显示

## 访问方式

### 电脑访问
- 直接在电脑浏览器中访问：http://localhost:5000

### 手机访问（小米15Pro优化）
1. 确保手机和电脑连接**同一个WiFi网络**
2. 在手机浏览器中访问：http://[电脑IP地址]:5000
3. 或者扫描二维码快速访问

## 配置说明

### 网络配置
- 默认端口：5000
- 如果需要更改端口，编辑`kanban_enhanced.py`文件
- 查找：`socketio.run(app, host='0.0.0.0', port=5000, debug=False)`
- 修改port参数即可

### 数据存储
- 数据自动保存在`kanban_data.json`文件中
- 备份文件保存在`backups`目录
- 支持数据导入导出（JSON/CSV格式）

## 常见问题

### Q1: 手机无法访问
- 检查电脑防火墙是否允许端口5000
- 确保手机和电脑在同一WiFi网络
- 在电脑上运行`ipconfig`查看IP地址

### Q2: 依赖安装失败
- 尝试手动安装：`pip install -r requirements.txt`
- 使用国内镜像：`pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple`

### Q3: Python未安装
- 下载Python：https://www.python.org/downloads/
- 安装时勾选"Add Python to PATH"

### Q4: 端口被占用
- 关闭占用5000端口的程序
- 或修改看板系统的端口号
